Getting Started


Just start tftpd32.exe (probably you have started it already). You get the following window :



Tftpd32 has already started
  • a TFTP server
  • a DHCP server (listening mode only)
  • a Syslog Server

The Syslog server is already full fonctionnal, however you need at least to configure the TFTP directory (current directory is the default) and the DHCP server must be configured to answer requests.

To change the directory, either use the Browse button, or go into the settings window.
